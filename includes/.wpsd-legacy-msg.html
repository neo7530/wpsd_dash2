Unsupported Version of WPSD Installed

You are running a version of WPSD that is now unsupported. This means that you had installed WPSD over an existing Pi-Star installation, which is now end-of-life and is not updated any longer.

To continue to receive updates and to take advantage of new WPSD features, you MUST install WPSD using a proper disk image for your hardware. Failing to do this will also result in a system with no updates and no support..
By January 1, 2024, your legacy installation will not receive any updates, and may result in an installation that does not work properly.

Before upgrading using a disk image, make a backup of your configuration which you can restore once the disk image installation is completed.

(This message does not exist once you upgrade using a disk image.)
